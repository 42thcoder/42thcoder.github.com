RocketMQ 如何保障高可用

2021-03-30 1 min read # 中间件 # RocketMQ

实践

官方提供了 Docker 镜像, 本地可以很方便地运行一个集群.

git clone https://github.com/apache/rocketmq-docker.git

开发者有两个选择, 自己编译镜像或者使用官方提供的, 我们直接使用官方的就好.

sh stage.sh 4.5.0 # 选择一个版本号
cd stages/4.5.0 
./play-docker-compose.sh

docker ps -a 看一下, 默认会启动一个两个 Broker 和一个 NameServer
